itle: Football Recruitment Analytics Dashboard
Description: An interactive Power BI dashboard designed to identify undervalued football players using performance metrics and market value analysis.

Key Features:

Dynamic Recruitment Score: A custom DAX model weighting performance (70%) and market value (30%) with a linear age bonus.

Market Efficiency Analysis: Scatter chart visualization to identify "Outliers" (High performance, Low cost).

Interactive Filters: Ability to search by budget, position, and minimum recruitment score.

Technical Stack:

Power BI Desktop

DAX (Data Analysis Expressions)

Data Cleaning & Transformation (Power Query)
